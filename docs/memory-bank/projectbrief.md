# Project Brief

## Overview
Infrastructure tooling to create and monitor Zcash infrastructure in public clouds.

## Core Features
- Terraform/Opentofu tooling for Zcashd for GCP
- Terraform/Opentofu tooling for Zebra for GCP
- GCP Stackdriver support for remote logging/monitoring

## Target Users
DevOps personnel

## Technical Preferences (optional)
- GCP
- Terraform/Opentofu, specifically leveraging the [Cloud Foundation Fabric](https://github.com/ZcashFoundation/cloud-foundation-fabric)
- Git
- GCP Stackdriver
